
# Macro & Markets Explorer

This is an educational Streamlit web app that demonstrates how macroeconomic factors affect different stock market sectors.

## Features
- Pre-set macroeconomic scenarios
- "Build your own" scenario with sliders
- Sector impact scoring model
- Automatic explanations of macro factors

## How to Run
1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the app:
   ```
   streamlit run app.py
   ```

3. The app will open in your browser.

## Purpose
Designed for high school DECA/business students to learn macroeconomics interactively.
